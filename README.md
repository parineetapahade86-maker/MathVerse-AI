# MathVerse AI

> Built solo by Parineeta, 16-year-old from Bhopal, India — for Gemini XPRIZE 2025

I hate memorizing formulas without seeing them. So I built MathVerse.

## What Works Right Now (100% my code)
- Visual step-by-step solver using Math.js
- 3D graph lab (Three.js) - rotate functions with mouse
- Voice input
- Single-file offline app (index.html)

## What Is NOT Working Yet
- AI Teacher is simulated only. I did not integrate Gemini API due to time.
- No real LLM calls in this build.
